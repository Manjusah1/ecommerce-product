# 🛍️ Dynamic E-commerce Product UI (Angular)

A modern, responsive **frontend-heavy e-commerce interface** built with Angular, focusing on product browsing, filtering, and smooth user experience.

---

## 🚀 Features

* 🧾 Product listing (grid & list view)
* 🔍 Real-time search with debounce
* 🎯 Advanced filtering (category, price, rating)
* ↕️ Sorting (price, popularity, newest)
* 📦 Product detail page with image gallery
* 🛒 Dynamic cart sidebar with quantity control
* ⚡ Reactive UI using RxJS
* 📱 Fully responsive design
* ⏳ Skeleton loaders for better UX

---

## 🛠️ Tech Stack

* Angular + TypeScript
* Angular Material
* RxJS
* HTML + SCSS
* Static JSON / Mock API

## ▶️ Getting Started

```bash
# Install dependencies
npm install

# Run the app
ng serve
```

Open: `http://localhost:4200`
